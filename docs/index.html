Work Education

Project Showcase

Average travel time to AIFA (Felipe Ángeles International Airport)

This small project consisted in creating connections with Google Maps Api to get routing requests with travel times from each of the municipalities in Mexico City to the recently built Felipe Ángeles International Airport (AIFA).

Proyecto pequeño en donde se utilizó la conexión con la API de Google Maps para realizar consultas de tiempos de viaje desde las alcaldías de la Ciudad de México hacia el Aeropuerto Internacional Felipe Ángeles (AIFA).

Masters Dissertation

A project developed during my masters degree studies, focusing on geocomputational methods used for allocating traffic into unsruveyed minor roads using data from major roads, based on regression models.

El proyecto que desarrolle para mi tesis de maestría, enfocado en métodos geo-computacionales utilizados para asignar datos de aforo a caminos secundarios sin conteos utilizando datos de caminos primarios, basado en modelos de regresión.

Work was developed for the Isle of Wight, UK, which offers good control over variables and has an enclosed road network.
